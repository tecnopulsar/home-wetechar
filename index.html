API REST

App Player proporciona una API RESTful completa para controlar todas sus funcionalidades de forma remota, organizada de manera modular.

Control de VLC

  • GET /api/vlc/status - Estado actual del reproductor
  • GET /api/vlc/play - Iniciar reproducción
  • GET /api/vlc/pause - Pausar reproducción
  • GET /api/vlc/stop - Detener reproducción
  • GET /api/vlc/playlist - Obtener playlist actual

Sistema

  • GET /api/system/info - Información del sistema
  • GET /api/system/resources - Recursos del sistema
  • GET /api/system/network - Información de red
  • GET /api/system/status - Estado del sistema
  • POST /api/system/reboot - Reiniciar el sistema

Archivos y Playlists

  • GET /api/files/list - Listar archivos disponibles
  • POST /api/files/upload - Subir archivos
  • DELETE /api/files/:id - Eliminar archivo
  • POST /api/playlist/upload - Subir listas de reproducción
  • GET /api/playlist/list - Listar playlists disponibles
  • DELETE /api/playlist/:id - Eliminar playlist

Aplicación

  • GET /api/app/version - Obtener versión de la aplicación
  • GET /api/app/config - Obtener configuración

Requisitos del Sistema

Hardware

  • Raspberry Pi 4 o superior
  • Mínimo 2 GB de RAM
  • Conexión a Internet

Software

  • Sistema operativo compatible (Raspbian, Ubuntu)
  • Node.js (versión 14 o superior)
  • npm (gestor de paquetes de Node.js)
  • VLC Media Player con interfaz HTTP

Tecnologías

  • Electron.js
  • Node.js
  • Express.js
  • Socket.IO
  • Python (scripts de control)
